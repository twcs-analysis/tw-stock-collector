## 📝 使用指南

### 如何更新進度

1. **完成任務時**
   - 在 DEVELOPMENT_SCHEDULE.md 中標記任務為完成 (✅)
   - 記錄實際工時

2. **階段完成時**
   - 更新 PROJECT_ROADMAP.md 中的階段狀態
   - 更新整體進度百分比

3. **每週回顧時**
   - 更新本週焦點與下週規劃
   - 記錄遇到的問題與解決方案
   - 調整下週時程安排
